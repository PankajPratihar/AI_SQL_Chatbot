# AI SQL Chatbot

A conversational SQL chatbot built using Streamlit, LangChain, Groq, and SQLAlchemy.

## Features

- Natural language to SQL
- Upload SQLite databases
- Custom database connection
- Modern Streamlit UI
- Chat history

## Installation

```bash


python -m venv venv
.\venv\Scripts\Activate.ps1
pip install -r requirements.txt
python -m streamlit run app.py
